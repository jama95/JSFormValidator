## Default modifiers

### Lower case

Read more about the modifier [here](/docs/modifiers.md#lower-case).

```html
<div class="fv-group">
  <label class="form-label">Lower case</label>
  <input type="text" name="lc" class="form-control" data-fv-modifiers="lower">
</div>
```
